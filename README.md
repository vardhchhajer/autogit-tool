# autogit-cli

Automate project documentation, GitHub publishing, and social media content generation

## Tech Stack

- TypeScript

## Installation

```bash
# Clone the repository
git clone https://github.com/USERNAME/autogit-cli.git
cd autogit-cli

# Install dependencies
npm install
```

## Usage

```bash
npm start
```
